# tecsus-frontend

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```

### Commit standardization

After preparing files to be committed by "git add ." or by IDE versioning control
```
npm run commit
```
